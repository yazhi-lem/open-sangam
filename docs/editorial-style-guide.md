# Editorial Style Guide

## 1. Tamil Script

- Use Unicode Tamil script (U+0B80–U+0BFF) exclusively; no transliteration in verse display.
- Transliteration in UI labels follows the ISO 15919 standard (e.g., Kuṟiñci, Pālai).
- Fonts: **Noto Serif Tamil** for verse text; **Noto Sans Tamil** for UI labels and prose.

## 2. Translation Conventions

### Urai (Modern Tamil Prose)

- Write in clear, contemporary Tamil; avoid archaic constructions.
- Preserve proper nouns (king names, place names) in original Tamil.
- Break long Sangam lines into natural prose sentences.

### English Translation

- Aim for literary rather than literal translation.
- Preserve Tamil proper nouns in ISO 15919 transliteration on first occurrence, e.g., *Neytal*.
- Use present tense for nature imagery; past tense for narrative.

## 3. Glossary Entries

- **Root** (வேர்ச்சொல்): cite the Dravidian Etymological Dictionary (DEDR) entry where possible.
- **Urichol** (உரிச்சொல்): use the classification from Tolkāppiyam.
- **Etymology**: one or two sentences maximum.
- **Gloss**: a single brief English phrase.

## 4. Cultural Notes

- Limit to 100–250 words per note.
- Always cite at least one reference (book, journal article, or DEDR entry).
- Categories: `dynasty`, `geography`, `flora`, `fauna`, `ritual`, `material-culture`, `other`.

## 5. Consistency Checklist

- [ ] Proper nouns spelled consistently throughout the corpus
- [ ] AI translations reviewed against at least one print edition
- [ ] Cultural notes peer-reviewed by a domain expert before publishing
